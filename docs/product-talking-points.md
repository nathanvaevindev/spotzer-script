## Website KSP (shorter version)
The website also adds value in AI search. The future brings new developments and we are ready for them.

We place an **LLM on the sitemap**: the LLM reads a text file across the web, and we create highly contextual value around it. It immediately signals what the site is about, so the AI does not have to scan everything — but pulls in data more easily and in richer form during the first scan.

This creates an enormous boost in visibility.
